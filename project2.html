Model Instructions

  • w, a, s, d: Rotate the camera. w = up, s = down, a = left, d = right (caps lock off)
  • Up Arrow + Down Arrow: Move the camera forward, and backward
  • j, l: Move the user-controlled ball (j=left, l=right)
  • Reset Camera: Returns the camera to its default position
  • Apply New Terrain Roughness: Regenerates the terrain with the current roughness setting
  • Roughness (slider): Set a new roughness for the terrain - then click on Apply New Terrain Roughness

Note

  • ball visibility: The ball is in a valley initially - press j to move it so you may start seeing it over peaks

Terrain Roughness (Editable): 60
